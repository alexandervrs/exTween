# exTween

exTween is a small tween framework that allows animation with easing of any built-in instance property plus any custom variables

* Identify your tweens with custom names and make them persist
* Animate any property, even custom ones
* Choose from a wide array of Easing algorithms
* Create quick local tweens that auto-cleanup on end
* Sync the tween playback to delta_time
* Execute scripts with arguments on Tween Complete
